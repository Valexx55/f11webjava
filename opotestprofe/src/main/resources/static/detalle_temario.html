Etiquetas Principales de HTML
ETIQUETA DESCRIPCIÓN ENLACE
h1 Para el título principal de la página (encabezado)
p Para definir párrafos de texto
a Para crear enlaces
img Para dibujar imágenes
ul Para definir una lista desordenada (estructura)
li Para definir un elemento de una lista
table Para crear una tabla de datos
div Para definir una sección genérica o contenedor
nav Para la sección de navegación (menú)
form Para la creación de formularios
